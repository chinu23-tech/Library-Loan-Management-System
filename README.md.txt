# Library Loan Management System

## Technologies Used
- Java
- JDBC
- Apache Derby
- Eclipse IDE

## Features
- Register Member
- Add Book
- Process Loan
- Return Book
- Transaction Management
- Rollback and Savepoint
- Performance Benchmarking

## Database Tables
- Members
- Books
- Loans

## Performance Tests
- Individual Insert vs Batch Insert
- Statement vs PreparedStatement

## How To Run
1. Run DatabaseSetup.java
2. Run InsertSampleData.java
3. Run MainApp.java
4. Run PerformanceTest.java
